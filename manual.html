S-CRM / 5. 계약서 작성 가이드


1. [Tab] 계약서 작성 페이지로 진입합니다.

Step1. 계약서 정보 작성

2. 프로젝트 / 거래선 정보를 확인합니다.

3. 영업 주체를 확인합니다.
기본값 : [Tab] 견적 작성에서 지정한 영업 주체

4. 담당자 변경 버튼을 클릭하여,
영업주체를 변경할 수 있습니다.

5. 선택한 담당자로 영업 주체가 변경되었습니다.

6. 계약 담당자를 확인합니다.
계약 담당자 : [Tab] 견적 작성에서 지정된 계약 담당자

※ 착공확정일 이내로, 수주이상 모니터링 항목들이 모두 해소되어야 합니다.
수주이상 모니터링 항목은 [Tab] 수주이상 모니터링에서 확인 가능합니다.
(수주이상 모니터링 항목 : 공동수급업체 미지정, 등)

7. 착공확정일을 입력/수정할 수 있습니다.
기본값 : [Tab] 견적 작성에서 입력받은 착공예정일

Tip! 계약서 유형

  • 전자계약 : S-CRM 내 전자서명으로 서명을 진행하는 계약
  • 종이계약 : 종이 계약서로 현장에서 협의 후, 서명이 완료된 계약
  • 비표준계약 : 고객의 계약서로 현장에서 협의 후 서명을 진행한 계약

8. 계약서 유형을 선택합니다.

  • 계약서 유형 : 전자계약 / 종이계약 / 비표준계약
  • 문서 유형 : 계약서 / 발주서

출력 문서가 필요한 경우 종이계약을 선택해주세요.
종이계약 선택 후 저장 시, 하단에 계약서 출력 버튼이 생성됩니다.

9. 완료일(납기일)을 수정할 수 있습니다.
기본값: [Tab]프로젝트/자재 선택에 입력된 납기예정일로 출력

10. 입력된 완료일(납기일) 값은 계약서 > 완료일(납기일)에 표기됩니다.

11. 고객 - 담당자 정보를 입력합니다. (선택 입력 값)

※ 소속(기본값 : 현장명) / 직위 / 담당자명(기본값 : 대표자명)

12. 입력된 값은 고객 정보 > 담당자, 담당자 정보에 표기됩니다.
담당자 : 담당자명 출력 / 담당자 정보 : 소속 + 직위 출력

13. 계약 일자를 수정할 수 있습니다.

14. NEGO 표기 여부를 확인합니다.
NEGO 표기여부 값은 견적작성 Tab에서 선택된 값으로 자동완성됩니다.

15. NEGO 표기 시,
계약서 > 상세 내역 > NEGO 값이 표기됩니다.

16. NEGO 미표기 시,
계약서 > 상세 내역 > NEGO 값이 표기되지 않습니다.

17. 공사명을 입력/수정합니다. (기본값 : 견적명)

18. 입력된 공사명은 계약서 > 계약정보 > 공사명에 출력됩니다.

현장명/현장 주소 입력 - 국가승강기번호 기준 불러오기
S-CRM 내 등록된 현장명/현장주소와 현재 사용 중인 건물명/건물 주소 표기가 상이할 경우, 국가승강기번호 불러오기를 이용하여 국가승강기번호 기준 건물명/소재지로 출력할 있습니다.
(국가승강기정보센터 링크 : https://www.elevator.go.kr/opn/MainPage.do)

19. (현장명 - 국가승강기번호 활용)

현장명(국가승강기번호 기준) 불러오기 버튼을 클릭하면,
현재 계약 작성 중인 프로젝트에 선택된 첫번째 호기의 국가승강기번호 기준
등록된 현장명으로 자동완성됩니다.
입력/수정된 현장명 값은 서명 문서에 출력됩니다.

20. (현장주소 - 국가승강기번호 활용)

현장주소(국가승강기번호 기준) 불러오기 버튼을 클릭하면,
현재 계약 작성 중인 프로젝트에 선택된 첫번째 호기의 국가승강기번호 기준
등록된 현장주소로 자동완성됩니다.

현장명, 현장주소 - 직접 입력/수정
서명 문서에 출력할 현장명, 현장주소 값을 직접 입력/수정할 수 있습니다.

21. (현장명 - 직접 입력/수정)

현장명을 입력/수정합니다.
입력/수정된 현장명 값은 서명 문서에 출력됩니다.

22. (현장 주소 - 직접 입력/수정)

주소 검색 버튼을 클릭 시, 주소 검색 팝업이 나타납니다.
주소 검색 팝업에서 주소를 검색하여, 선택합니다.

23. (현장 주소 - 직접 입력/수정)

상세 주소를 입력/수정할 수 있습니다.

24. 입력된 현장명/현장주소 값은 계약서 > 현장명/현장주소에 표기됩니다.

Step2. 청구 조건 입력/수정

견적 결재 미승인 시, 청구 조건을 입력이 불가합니다.
견적 결재 승인 완료 시, 청구 조건을 입력하실 수 있습니다.

25. 청구 정보를 확인 후, 입력/수정할 수 있습니다.

26. 대금 지급 방식을 선택합니다.
일괄 지급 방식 / 각자 지급 방식

27. 각자 지급 방식을 선택할 경우,
서명 문서에 '공동수급업체 계좌 정보' 별지가 첨부됩니다.

28. 세금 정보를 선택합니다.

29. 계산서 구분을 선택합니다.
선택한 값은 계약서/발주서 문서 기준, 대금지급 > 지급 조건에 출력됩니다.

청구 조건 > 계산서 발행 방식에는 총 2가지가 있습니다.

  1. 통합 발행
  2. 분할 발행

2-1. 통합 발행
지정된 시기에 전체 금액을 모두 일괄로 청구합니다.

30. 통합발행을 선택합니다.

31. 구분값을 선택 후, 청구 금액을 확인합니다.
※ 구분 : 계약금 / 중도금 / 완불금

32. 지급시기를 선택/입력합니다.
계약금일 경우, 지급시기 : 계약시 / 직접입력
중도금일 경우, 지급시기 : 주자재반입시 / 직접입력
완불금일 경우, 지급시기 : 계약시 / 주자재반입시 / 직접입력

33. 발행일을 지정합니다. 우측 하단 청구금액 합계를 확인할 수 있습니다.

청구금액 합계 : 계산서 발행 시 청구될 금액, (통합발행) 계산서 발행 > 금액과 동일

2-2. 분할 발행
지정된 시기에 금액을 분할하여 청구합니다.

34. 분할 발행을 선택합니다.
분할 발행일 경우, 계약금 / 중도금or완불금 / 완불금로 구분하여 금액을 청구할 수 있습니다.

금액 분할 방식에는 총 2가지가 있습니다.

  1. 비율 입력
  2. 금액 입력

분할 방식 - 비율 입력

35. (분할 방식 선택)
비율 입력을 선택합니다.

비율 입력시 주의사항

  • 중도금or완불금은 필수값입니다.
    2회 발행 시, 계약금 & 중도금or완불금 입력란에 비율을 입력해주세요.
  • 각 시기별 비율의 합이 100이 될 수 있도록 입력해주세요.

※ 청구 금액 합계 = 계약금 + 중도금(또는 완불금) + 완불금

입력된 청구 금액 합계는 반드시 계약 금액(부가세 별도)와 일치해야 합니다.
청구 금액 합계가 계약 금액과 다르게 입력되지 않도록 주의해주세요!

36. 비율 입력란이 활성화됩니다.
비율을 입력하여, 각 시기별 청구 금액을 조정합니다.

각 시기별 비율의 합이 100이 될 수 있도록 입력해주세요.

37. 지급시기를 선택/입력합니다.
계약금일 경우, 지급시기 : 계약시 / 직접입력
중도금일 경우, 지급시기 : 주자재반입시 / 직접입력
완불금일 경우, 지급시기 : 계약시 / 주자재반입시 / 직접입력

38. 각 구분별 발행일을 지정합니다.

분할 방식 - 금액 입력

39. 금액 입력을 선택합니다.

※ 청구 금액 합계 = 계약금 + 중도금(또는 완불금) + 완불금

입력된 청구 금액 합계는 반드시 계약 금액(부가세 별도)와 일치해야 합니다.
청구 금액 합계가 계약 금액과 다르게 입력되지 않도록 주의해주세요!

40. 금액 입력란이 활성화됩니다.
금액을 직접 입력하여, 각 시기별 청구 금액을 조정합니다.

41. 좌측 비율 입력란에는
입력된 금액에 맞춰, 전체 청구금액 대비 입력된 금액의 비율이 출력됩니다.

42. 우측 하단 청구금액 합계를 확인합니다.
계약 금액(부가세 별도)과 동일한 값으로 출력되는지 확인해주세요.

청구금액 합계 : 계산서 발행 시 청구될 금액,
(분할발행) 계산서 발행 > 계약금 + 중도금or완불금 + 완불금 합계

43. 입력된 지급시기 / 지급비율 / 금액 / 지급 조건은
계약서 내 대금지급에 출력됩니다.

44. 수금 항목을 선택합니다.

45. 발송 구분을 선택합니다.

46. 공사 유형과 계약 금액을 산정하여,
수리공사 기성매출 유형이 출력됩니다.

Step3. 계약 요약사항/특이사항/내부용 메모 입력

Tip!

  • 계약 요약사항 : 간편 서명(인증서 서명) 진행 시, 고객이 이해하기 쉽도록 계약서 내용에 대한 간단한 요약 정보를 기재합니다.
  • 특약사항(계약용) : 해당 계약의 특이사항을 기재합니다.
  • 내부용 메모 : S-CRM 임직원 내에서만 확인할 수 있는 내부용 메모를 기재합니다.

47. 계약 요약사항을 입력합니다.
입력된 계약 요약사항은 간편 서명(인증서 서명) 진행 시,
모바일 계약서에 표기됩니다.

48. 입력된 계약 요약사항은 모바일 계약서 > 계약 요약사항에 출력됩니다.

49. 특약사항(계약용)을 입력합니다.
입력된 특약사항(계약용)은 계약서 > 특약사항에 표기됩니다.

50. 입력된 특약사항은 계약서 > 계약 특약사항에 출력됩니다.

51. 내부용 메모를 입력합니다.
내부용 메모는 S-CRM 내에서만 확인할 수 있습니다.

Step4. 저장 & 계약서 미리보기 & 계약서 출력

52. 저장 버튼을 클릭합니다.
입력된 값이 저장되었습니다.

견적 결재 완료 시, 저장된 계약서 정보로 서명을 요청할 수 있습니다.
서명 방법에는 전자 계약 / 종이 계약 총 2가지 케이스가 있습니다.

  1. 계약서 유형 > 전자 계약일 경우, [Tab]서명 요청에서 서명을 진행합니다.
    전자 계약에 대한 자세한 설명은 S-CRM / 전자 계약 가이드를 참고해주세요.
  2. 계약서 유형 > 종이 계약일 경우, 계약서 출력 버튼이 나타납니다.
    종이 계약에 대한 자세한 설명은 S-CRM / 종이 계약 가이드를 참고해주세요.

(비표준 계약)

  1. 계약서 유형 > 비표준 계약일 경우, 서명 완료 문서를 업로드합니다.
    서명 완료 문서 업로드에 대한 자세한 설명은 S-CRM / 종이 계약 가이드 > Step3. 서명 완료 문서 제출을 참고해주세요.

53. 계약서 유형 > 전자 계약일 경우, [Tab] 서명 요청 진입이 가능합니다.

54. 다음 버튼을 눌러 [Tab] 서명 요청으로 이동합니다.
전자 계약에 대한 자세한 설명은 S-CRM / 전자 계약 가이드를 참고해주세요.

55. 계약서 유형 > 종이 계약일 경우, 계약서 출력 버튼이 나타납니다.
종이 계약에 대한 자세한 설명은 S-CRM / 종이 계약 가이드를 참고해주세요.

56. 계약서 유형 > 비표준 계약일 경우, 서명 완료 문서를 업로드합니다.
서명 완료 문서 업로드에 대한 자세한 설명은 S-CRM / 종이 계약 가이드 > Step3. 서명 완료 문서 제출을 참고해주세요.
